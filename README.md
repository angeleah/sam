SAM - Salon Assistant Mini
==========================

Usage:
------
- To run the app:
  + Clone the repo
  + $ bundle
  + Run $ rake db:migrate
  + Run $ rake db:test:prepare
  + Type $ rails s (or rails server)
  + Open a browser and navigate to localhost:3000
  + When asked to authorize, type in  username: user, password: Clients

- To run the specs:
  + Clone the repo
  + $ bundle
  + Run $ rake db:migrate
  + Run $ rake db:test:prepare
  + Run $ rspec
